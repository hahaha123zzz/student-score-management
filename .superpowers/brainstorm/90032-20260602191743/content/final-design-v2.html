⚙️ 编译 & 运行（不需要CMake）

一行编译，双击运行
// 编译（在项目根目录执行） g++ -std=c++17 -I include -I lib -o edugrade.exe src/main.cpp src/util.cpp src/auth.cpp src/student.cpp src/exam.cpp src/grade.cpp src/stats.cpp -lws2_32 // 运行 edugrade.exe // 浏览器打开 http://localhost:8080
